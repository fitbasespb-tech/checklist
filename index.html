Роль 01

Путь администратора

Ежедневная работа на ресепшене: клиенты, абонементы, расписание, продажи

01
?
Что-то непонятно — пишите менеджеру. Лучше уточнить сейчас, чем разбираться на старте работы с реальными клиентами.
1

Работа с клиентом

  • Создать нового тестового клиента из раздела:
    • заявки
    • клиенты
    • занятие в расписании
  • Оставить комментарий в карточке клиента
  • Поставить задачу на клиента
  • Начислить бонусные баллы
  • Добавить тег клиенту
Дополнительно
  • Написать клиенту в мессенджер
2

Работа с абонементами

  • Продать абонемент тестовому клиенту
  • Отредактировать абонемент:
    • количество посещений
    • срок действия
    • количество заморозок
  • Заморозить и разморозить абонемент
  • Сделать возврат
Дополнительно
  • Провести внутреннюю рассрочку
3

Работа с расписанием

Если проход через ресепшен — запустите клиента через ресепшен

  • Записать клиента с активным абонементом в групповое занятие
  • Поставить статус «Пришёл»
  • Изменить статус клиента на «Отказался»
  • Записать клиента на другое занятие / другой день
  • Проверить:
    • историю посещений клиента
    • записи клиента и статусы
    • корректное списание посещений
4

Работа с персональными тренировками

  • Записать клиента на персональную тренировку
  • Провести оплату услуги
  • Поставить статус «Пришёл» в занятии
  • Удалить запись и найти её в архиве
Дополнительно
  • Оформить услугу на другого клиента
5

Работа с заявками

  • Переместить клиента на нужный этап воронки
  • Переместить клиента в «Отказ» и выбрать причину
  • Просмотреть аналитику заявок
Дополнительно
  • Внести бюджет и тег у заявки

Роль 04

Путь маркетолога

Промокоды, рассылки, автоматизации, удержание и проверка отчётов

04
  • Создать промокод
  • Создать скидку
  • Настроить автоматическую скидку для группы клиентов
  • Создать тестовую push-рассылку — отправить только себе

    • бонусам

Роль 05

Путь руководителя

Контроль клуба или студии через дашборды, отчёты и проверку ключевых процессов

05
  • Изучить отчёты «Рабочего стола»
  • Проверить:
    • отчёты по продажам
    • посещаемость клуба
    • необработанные заявки
    • неотвеченные сообщения в ФБЧ
  • Проверить заполняемость занятий
  • Проверить эффективность сотрудников:
    • тренеров
    • администраторов
    • менеджеров по продажам
  • Проверить продления абонементов
  • Проверить работу автоматизаций
  • Проверить корректность и функциональность:
    • мобильного приложения клиента
    • личного кабинета клиента
    • приложения тренера
  • Проверить аналитику работы с заявками
  • Проверить основные бизнес-процессы клуба:
    • запись клиента
    • проведение оплаты
    • списание посещений
    • возврат
    • расчёт зарплаты тренеров
Дополнительно
  • Проверить онлайн-оплаты и поступление ДС на счёт:
  • — через приложение
  • — через ЛК клиента в расписании
  • — по прямой ссылке через мессенджер
